Идея
1.1 Жанр

2D платформер с элементами головоломки и управления временем.

1.2 Игровой мир

Игровой мир — это разрушенная станция управления временем.

Особенности мира:

разделён на “сектора времени” (прошлое, настоящее, будущее)
одни и те же локации выглядят по-разному в разных временных состояниях
некоторые объекты существуют только в определённом времени

Изначально мир:

нестабилен
содержит разрывы времени (гличи)
части уровней недоступны

По мере прохождения:

игрок стабилизирует временные зоны
получает доступ к новым состояниям времени
изменяет окружение через прошлое

На поздних этапах:

временные линии начинают пересекаться
появляются парадоксы (объекты дублируются, исчезают)
уровни становятся нелогичными и сложными
1.3 Сюжет

Игрок — оператор станции, который потерял контроль над системой времени.

Сюжет подаётся через:

записи предыдущих операторов
визуальные изменения уровней
“эхо” прошлых действий игрока

Ход сюжета:

герой просыпается на разрушенной станции
обнаруживает способность переключать время
пытается восстановить систему
сталкивается с последствиями изменений (парадоксы)
понимает, что сам стал причиной катастрофы

Финал:

выбор: полностью остановить время или зациклить его

Основная тема:

контроль vs последствия
ошибки, которые нельзя отменить
1.4 Персонаж(-и) и враги
Главный персонаж

Оператор станции.

Особенности:

не разговаривает напрямую
его прошлые версии появляются как “эхо”
может взаимодействовать с самим собой из прошлого
Враги

Враги — временные аномалии.

Типы:

Эхо-двойники — повторяют действия игрока с задержкой
Разломы — области, где нарушена физика
Парадоксы — враги, появляющиеся при ошибках игрока

Особенность:

многие враги — результат действий игрока
